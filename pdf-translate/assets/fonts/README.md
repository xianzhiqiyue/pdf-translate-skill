# Optional bundled fonts

Put licensed `.ttf`, `.otf`, or `.ttc` files here if you want this skill package to carry fonts with it.

The script searches this directory before system font locations. For Chinese/Japanese/Korean output, common choices are:

- Noto Sans CJK / Noto Sans SC
- Source Han Sans

No font binary is bundled by default because CJK fonts are large and each project/distribution must verify font license and package-size tradeoffs.
